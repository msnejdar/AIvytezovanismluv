# AI Vytěžování Smluv

Projekt pro inteligentní vyhledávání a analýzu smluv pomocí AI.

## Instalace

```bash
cd AI-SEARCH
npm install
```

## Spuštění

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deployment
Automatický deployment na Vercel při každém push do main branch.
